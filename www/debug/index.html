Nightscout Clock

Debug Build - For testing and troubleshooting

This is a debug build with extra logging enabled. For normal use, install the release build instead.

Install debug firmware

  1. Power on the Ulanzi TC001 by holding the < and > buttons for 3 seconds
  2. Connect it to this computer with a USB-C cable
  3. Click the Install button below and select the serial port

Ready? Plug in your clock and click Install.

Your browser does not support Web Serial.
Please open this page in Google Chrome or Microsoft Edge on a desktop computer.
